    Tutorials: Onboarding

    Welcome to the Rendersynth tutorials. This section guides you through building synthetic datasets from setup to advanced generation workflows.

    Tutorial Structure

    This tutorial series is organized into a logical progression:

    1. Classes & Entities – Define your dataset schema with custom classes and multi-object grouping
    2. Create Pipeline – Build your first randomization pipeline with distributions and operations
    3. Labeling – Set up annotation formats and label generation strategies
    4. Distributions – Master probability distributions for scene parameters
    5. Create Labeling Formats – Define custom label export formats
    6. Shader Randomization – Randomize material properties and textures
    7. Common Pitfalls – Avoid gotchas and optimize your pipelines
    8. Matching Labeling and Formats – Align your pipeline output with label structure
    9. Batch Generation – Configure and run large-scale dataset renders
    10. Preview – Inspect outputs, bounding boxes, and statistics in real-time
    11. Importing and Exporting – Load configurations and export datasets
    12. Labeling Rigs and Keys – Advanced labeling techniques and rig management

    Recommended Learning Paths

    For Beginners

    Start simple and build understanding: 1. Classes & Entities – Understand the data model 2. Create Pipeline – Build your first pipeline 3. Labeling – Add output labels 4. Batch Generation – Render your first dataset

    For Intermediate Users

    Dive into the core features: 1. Distributions – Advanced randomization 2. Shader Randomization – Material variation 3. Matching Labeling and Formats – Alignment 4. Preview – Quality control

    For Advanced Users

    Optimize and extend: 1. Common Pitfalls – Best practices 2. Create Labeling Formats – Custom export formats 3. Labeling Rigs and Keys – Advanced setups 4. Importing and Exporting – Version control and sharing
